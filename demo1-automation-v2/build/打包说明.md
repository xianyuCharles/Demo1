# Demo 1 打包说明

## 交付标准
客户零配置开箱即用，产品包含运行环境。

## 打包方式：本地打包

### 前置条件
- Windows 10/11 电脑
- Python 3.10+（仅打包时需要，交付给客户不需要）
- 网络连接（首次安装依赖）

### 打包步骤

#### 方式一：一键打包（推荐）
1. 双击运行 `build.bat`
2. 等待打包完成（1-3分钟）
3. 交付包生成在 `交付包/` 目录

#### 方式二：手动打包
```bash
# 1. 安装依赖
pip install pyinstaller pandas openpyxl jinja2

# 2. 执行打包
pyinstaller --onefile --name "订单数据处理工具" --add-data "modules;modules" --hidden-import pandas --hidden-import openpyxl --hidden-import jinja2 --console scripts/process.py

# 3. 准备交付包
mkdir 交付包
copy dist\订单数据处理工具.exe 交付包\
xcopy /e input 交付包\input\
mkdir 交付包\output
```

### 交付包结构
```
交付包/
├── 订单数据处理工具.exe   ← 主程序（含运行环境）
├── input/                 ← 客户放入CSV文件
│   └── 订单数据.csv       ← 示例数据
└── output/                ← 程序输出目录
    ├── 清洗明细.csv
    ├── 每日汇总.csv
    ├── 品类汇总.csv
    ├── 渠道汇总.csv
    ├── 地区汇总.csv
    ├── 日报看板.html
    ├── 企微通知文案.txt
    ├── 钉钉通知文案.txt
    └── 异常清单.txt
```

### 客户使用方法
1. 将订单CSV文件放入 `input` 文件夹
2. 双击运行 `订单数据处理工具.exe`
3. 查看 `output` 文件夹中的结果
4. 打开 `日报看板.html` 查看可视化报表

### 注意事项
- exe文件大小约50-80MB（包含Python运行环境）
- 首次运行可能需要3-5秒解压
- 支持Windows 10/11 64位系统
- 无需安装Python或其他依赖

### 后续自动化（可选）
