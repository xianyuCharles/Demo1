# Demo 1: 订单数据自动化处理系统

## 📦 产品介绍

这是一个**报表自动化样例**。客户每天只需把订单后台导出的CSV放进文件夹，运行程序后自动完成：

- ✅ 数据清洗（重复订单、缺失手机号、异常金额）
- ✅ 销售汇总（每日/品类/渠道/地区）
- ✅ 可视化日报看板（HTML）
- ✅ 企微/钉钉通知文案生成

**原来1-2小时的表格整理工作，压缩到几分钟。**

## 🚀 使用方法

### 第一步：放入数据
将订单CSV文件放入 `input/` 文件夹：
```
input/
└── 订单数据.csv    ← 把文件放这里
```

### 第二步：运行程序
双击运行 `生成日报.exe`（Windows）或执行脚本（开发环境）。

### 第三步：查看结果
处理完成后，结果自动保存在 `output/` 文件夹：
```
output/
├── 清洗明细.csv        # 清洗后的完整数据
├── 每日汇总.csv        # 按日期统计
├── 品类汇总.csv        # 按品类统计
├── 渠道汇总.csv        # 按渠道统计
├── 地区汇总.csv        # 按地区统计
├── 日报看板.html       # 🌟 可视化看板（用浏览器打开）
├── 企微通知文案.txt    # 复制粘贴到企业微信
├── 钉钉通知文案.txt    # 复制粘贴到钉钉
└── 异常清单.txt        # 数据问题清单（如有）
```

## 🔍 自动检测的数据问题

系统会自动检测并报告以下问题：
- ❌ **重复订单** - 相同订单号的重复记录
- ❌ **无效手机号** - 非11位或非1开头的手机号
- ❌ **异常金额** - 金额小于0或超过100万
- ❌ **空行数据** - 完全为空的行
- ⚠️ **缺失字段** - 必填字段为空

## 🛠️ 开发环境运行（技术人员）


## 🆘 常见问题

**Q: CSV文件编码不对，显示乱码？**
A: 请用Excel另存为"CSV UTF-8（逗号分隔）"格式。

**Q: 提示"未找到CSV文件"？**
A: 请确认CSV文件已放入 `input/` 文件夹。

**Q: 生成的看板打不开？**
A: 请用浏览器（Chrome/Edge/Firefox）打开 `output/日报看板.html`。
